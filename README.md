CineMind-AI 🎬

An intelligent movie recommendation platform built with React.js, Redux Toolkit, and the TMDB API — helping users discover, search, and explore movies with a fast, personalized experience.

Features


🔍 Smart Search & Discovery — search and browse movies using real-time data from the TMDB API
🔐 Secure Authentication — user sign-up/login via Firebase Authentication
🛡️ Protected Routing — restricted routes ensure only authenticated users access personalized content
⚡ Optimized Performance — custom React hooks, lazy loading, and memoization for a smooth experience
🗂️ Centralized State Management — Redux Toolkit manages global app state (auth, search, movie data)
🌐 Efficient API Handling — debounced/optimized API calls to TMDB to minimize redundant requests
